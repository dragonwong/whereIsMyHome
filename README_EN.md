# Where Is My Home

This project aims to help me to find my home in Beijing.

## Tutorial

### Install

```bash
npm install
```

### Some Modify

In `src/pages/find/index.html`, replace `{{your_key}}` with your own key from https://lbs.amap.com/dev/key/app# .

### Start

```
npm start
```

Visit http://localhost/home/find .
